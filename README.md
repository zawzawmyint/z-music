# Z-Music

Z-Music is a sleek and modern music player built with React Native and Expo. It's designed to provide a seamless and enjoyable listening experience, allowing you to enjoy your favorite tracks anytime, anywhere.

## Features

- **Modern UI**: A clean and intuitive user interface that makes navigation and music playback effortless.
- **Local Music Playback**: Play audio files directly from your device's storage.
- **Now Playing Screen**: A dedicated screen that displays the currently playing track, with controls for play, pause, skip, and seek.
- **Playlist Management**: Create and manage playlists to organize your music library.
- **Responsive Design**: The app is designed to look great on a wide range of screen sizes and orientations.

## Tech Stack

- **Framework**: React Native with Expo
- **Navigation**: Expo Router
- **UI Components**: Expo Vector Icons, React Native Gesture Handler, React Native Reanimated
- **Audio Playback**: Expo AV
- **File System Access**: Expo File System, Expo Media Library
- **Styling**: React Native StyleSheet with `react-native-size-matters` for responsive scaling

## Installation

To get started with Z-Music, follow these steps:

1. **Clone the repository**:

   ```bash
   git clone https://github.com/zawzawmyint/z-music.git
   cd z-music
   ```

2. **Install dependencies**:

   ```bash
   npm install
   ```

3. **Start the development server**:

   ```bash
   npm start
   ```

   - Scan the QR code with the Expo Go app on your Android or iOS device.
   - Alternatively, you can run the app on the web by selecting "w" in the terminal.
